# 守护智慧稻田

面向 1-5 年级小学生的农业科普互动游戏。玩家在真实稻田场景中观察水稻五个生长阶段，识别随机出现的病虫害，并选择正确工具完成防治。

## 项目状态

当前子项目包含：

- 游戏设计文档：[docs/DESIGN.md](docs/DESIGN.md)
- 静态网页项目骨架：`index.html`、`styles.css`、`app.js`
- 可本地预览的第一版交互壳

## 本地运行

这个项目不依赖构建工具，可以直接打开 `index.html`，也可以用本地静态服务器预览：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000
```

## 核心规则

- 水稻阶段依次为：插秧期、分蘖期、拔节期、抽穗期、成熟收割期。
- 每局从 6 种病虫害中随机出现 3 种。
- 玩家先选择工具，再点击病虫害。
- 工具正确时清除目标；工具错误时提示“工具不对，请重新选择”。
- 3 个目标全部清除后通关。
